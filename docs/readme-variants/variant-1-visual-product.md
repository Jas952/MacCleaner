# Variant 1: Visual Product Page

Best when the repository should feel like a polished app landing page.

Structure:

1. Hero image: `docs/readme-media/hero.png`
2. Centered app title and short agent-oriented description
3. For-the-badge stack row: Swift, macOS, Xcode, Codex, Gemini
4. Slideshow GIF for the first visual impression
5. Collapsible full gallery with `showcase-grid.png` and `feature-cards.png`
6. Build, install, repository layout, distribution notes
7. Contact block

Visual signature:

```md
<p align="center">
  <img src="./docs/readme-media/hero.png" alt="MacCleaner hero" />
</p>
```

This is the variant implemented in the root README.
