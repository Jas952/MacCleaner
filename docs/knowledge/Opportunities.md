# Незавершённые возможности MacCleaner

В этом файле оставлены только те IMP, над которыми ещё нужно работать. Завершённые и удалённые направления намеренно не дублируются, чтобы документ был рабочим списком дальнейших задач.

Статусы:

- **Требует доработки** — пользовательский сценарий уже существует, но исходная задача закрыта не полностью.
- **Стоит сделать** — к реализации полноценного сценария ещё не переходили.

## Стоит сделать

| IMP | Что это значит простыми словами | Что конкретно нужно сделать | Зачем это нужно / результат | Приоритет |
| --- | --- | --- | --- | --- |
| **IMP-004** | Большой процент занятой RAM сам по себе не означает проблему. | Безопасные правила завершены: приложения не закрываются автоматически, `purge` не используется, выбор остаётся ручным, inactive/compressed/wired memory показываются информативно. | Подключить реальные системные memory-pressure и swap-метрики macOS и использовать их вместо одной классификации по `usedPercent`. | Критический |
| **IMP-008** | Пользователь сможет временно управлять вентиляторами и вернуть Auto. | Проверить надёжный hardware API, manual mode, безопасный возврат в Auto и различия Intel/Apple Silicon. | Контролируемое охлаждение без зависшего ручного режима. | Высокий |
| **IMP-052** | Не давать Mac уснуть в понятных пользовательских режимах. | Сделать профили: экран, вся система, таймер, презентация, пока работает приложение, только питание от сети. | Предсказуемый sleep prevention вместо одной неясной кнопки. | Высокий |
| **IMP-053** | Безопасно открыть системную настройку лимита заряда, если Mac её поддерживает. | Capability check для macOS/Apple Silicon и переход в системные настройки 80–100%; не использовать private SMC keys. | Безопасное управление зарядом без неофициального вмешательства в SMC. | Высокий |

### Работа с AI-агентами

| IMP | Что это значит простыми словами | Что конкретно нужно сделать | Зачем это нужно / результат | Приоритет |
| --- | --- | --- | --- | --- |
| **IMP-041** | Поддержку AI-агентов нужно сделать расширяемой, а не набором отдельных эвристик. | Вынести adapter registry: install anchors, executable/config roots, capabilities и fixtures против ложных совпадений. | Проще добавлять новых агентов и точнее определять владельца процессов. | Высокий |
| **IMP-043** | Сравнивать агентов на одинаковой безопасной тестовой задаче. | Создать Native Agent Arena с изолированными fixtures и метриками correctness, duration, CPU/RSS, tokens/cost. | Объективное сравнение агентов без запуска рабочих проектов. | Высокий; после IMP-041 |
| **IMP-044** | Показать связи между агентом, дочерними процессами, MCP и инструментами. | Построить event graph с PID, временем, источником доказательства и уровнем уверенности. | Понятно, откуда реально взялась нагрузка. | Высокий |
| **IMP-045** | Не приписывать агенту всю нагрузку системы. | Отделять основной PID, descendants, MCP/helpers, shared и unknown load. | Точная атрибуция нагрузки конкретной сессии. | Высокий; после IMP-044 |
| **IMP-056** | Связать workspace, IDE, агента, language server, build tool, MCP, контейнер и indexer. | Создать Developer Workload Graph с evidence, timestamp и confidence. | Объяснять происхождение developer-нагрузки, а не просто показывать список процессов. | Высокий |
| **IMP-058** | По запросу получать глубокий профиль runtime агента. | Сделать opt-in adapters для Memray, heap snapshots, MemLab и Node lifecycle diagnostics на изолированных fixtures. | Находить утечки и причины роста памяти без постоянного профилирования. | Высокий; после отдельного fixture-потока |
| **IMP-059** | Дать агентам безопасный read-only способ запросить своё состояние. | Локальный MCP/API с redaction, workload graph и profiler reports; destructive tools выключены по умолчанию. | Агент сможет объяснять своё состояние, не получая права удалять или завершать процессы. | Высокий; после IMP-056/058 |
| **IMP-060** | Рекомендации должны быть доказательными и проверяемыми. | Показывать evidence, confidence, основной вклад, обратимое действие и повторный замер результата. | Пользователь понимает, почему предложено действие и помогло ли оно. | Высокий; после IMP-056/058 |
